## Getting Started

1. **Setup Environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **Configuration**:
   Copy `.env.example` to `.env` and adjust the settings:
   ```bash
   cp .env.example .env
   ```

3. **Run Application**:
   ```bash
   python run.py
   ```
